  How can I create a data frame in Python?

[AI]: You can create a data frame in Python using the pandas library. Here's an example of how to create a data frame:

1. Import the pandas library:

```python
import pandas as pd
```

2. Create a dictionary containing your data:

```python
data = {'Name': ['John', 'Emma', 'Peter', 'Olivia'],
        'Age': [25, 28, 30, 35],
        'City': ['New York', 'London', 'Paris', 'Tokyo']}
```

3. Convert the dictionary into a pandas data frame:

```python
df = pd.DataFrame(data)
```

4. Print the data frame:

```python
print(df)
```

Output:

```
    Name  Age      City
0   John   25  New York
1   Emma   28    London
2  Peter   30     Paris
3  Olivia  35     Tokyo
```

Alternatively, you can read data from a file (such as a CSV, Excel, or
SQL database) using pandas functions like `pd.read_csv()`,
`pd.read_excel()`, or `pd.read_sql()`.

Remember to install pandas if you haven't already by running `pip
install pandas` in your terminal.
